Link to profiles of the most popular social networks.

== Description ==
Create a widget that will display the profile links to different social networks. The plugin supports the most popular sites and their base color.

== Installation ==

1. Upload the `social-profiles` folder to the `/wp-content/plugins/` directory
2. Activate the Social Profiles plugin through the 'Plugins' menu in WordPress
